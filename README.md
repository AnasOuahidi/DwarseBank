#Dwarse Development
##Banque du projet transversale
###Lancer en local : 
Cloner le projet
```
git clone https://github.com/AnasOuahidi/DwarseServer.git
```
Installer les dépendances
```
composer update
```
Lancer le serveur
```
php bin/console server:run
```
